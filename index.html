mediasoup

Cutting Edge WebRTC Video Conferencing

$ npm install mediasoup

Powerful SFU

Due to its versatility and flexibility, the SFU model (Selective Forwarding Unit) for multiparty conferencing and similar scenarios is taking over the more rigid and resource hungry MCU model.

Node.js module

Instead of creating yet another opinionated server, mediasoup is a Node.js module which can be integrated into a larger application. mediasoup provides a low level API that enables different use cases up to your application.

Client side libraries

mediasoup comes with extremely powerful client side libraries (mediasoup-client JavaScript library and libmediasoupclient C++ library) that make it easy to build video conference applications that run in any browser or device by using an unified API.
